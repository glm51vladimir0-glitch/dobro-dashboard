# DOBRO Volunteer Dashboard

Интерактивный аналитический дашборд для мониторинга и прогнозирования добровольческой активности в России.

## Функционал
- 📊 Динамика волонтёров, регистраций и акций по годам
- 👥 Разбивка по возрастным группам с прогнозами
- 🗺️ Топ-15 регионов с фильтрацией
- 📈 Логистическая модель роста с демографическими потолками
- 🎯 Доверительные интервалы (±15%) на прогнозах
- 🔍 Фильтры: регион, акция, возрастные группы (multi-select)

## Технологии
- Chart.js (inline, работает из file://)
- Встроенные данные (window.INJECTED_DATA)
- Без внешних зависимостей — один HTML-файл
