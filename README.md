# DOBRO Volunteer Dashboard

Аналитический дашборд для мониторинга и прогнозирования добровольческой активности в России.

## Структура проекта

```
├── index.html              # Актуальная версия дашборда (v8.2)
├── data/                   # Исходные данные
│   ├── жмурко_ПД_все_время.csv       # Данные по ПД (55K строк, 25 акций)
│   ├── жмурко_реги_все_время.csv     # Данные по регионам (89 регионов)
│   └── forecast_data_new.json         # Сводные данные РФ
├── scripts/                # Скрипты обработки и генерации
│   ├── build_dashboard_data.py        # Генерация data blob
│   ├── build_dashboard_v4.py          # Сборка v4
│   ├── build_dashboard_v5.py          # Сборка v5
│   ├── build_dashboard_v6.py          # Сборка v6
│   ├── analyze.py                     # Анализ данных
│   ├── forecast.py                    # Прогнозирование
│   └── rosstat_demographics.py        # Демографические потолки
├── original/               # Исходный дашборд (от заказчика)
├── versions/               # История версий (v2-v8.2)
└── lib/                    # Chart.js
```

## Функционал
- 📊 Динамика волонтёров, регистраций и акций по годам
- 👥 Разбивка по возрастным группам с прогнозами
- 🗺️ Топ-15 регионов с фильтрацией
- 📈 Логистическая модель роста с демографическими потолками
- 🎯 Доверительные интервалы (±15%)
- 🔍 Фильтры: регион, акция, возраст (multi-select)
- 📑 Табы: Общий / По возрастам на каждом графике

## Как собрать
```bash
python3 scripts/build_dashboard_data.py
python3 scripts/build_dashboard_v6.py
```
